Restoring data
  1. Open Backup.
  2. Choose a data source. The default data source is the internal storage. Touch to select data from Cloud Drive or a USB storage device.
  3. Choose a data source. The default data source is the local storage. Touch to select data from Cloud Drive.
  4. Select the backup file you want to restore. If the backup file has been encrypted, enter the password when prompted.
  5. Select the data you want to restore, and follow the onscreen instructions to restore it.
