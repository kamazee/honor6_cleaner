Installing an application

With App Installer, you can install applications stored in your internal storage or microSD card.

  1. Open App Installer.
  2. Select the application you want to install.
  3. Touch > Install, and follow the onscreen instructions to install the application.

During the installation, an Install blocked message may be displayed. Read the message carefully. If you want to continue with the installation, follow the onscreen instructions to allow the installation of third-party applications.
